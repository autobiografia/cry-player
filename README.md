
# cry-player

This project was made to watch crunchyroll videos.
(BETA)

## Demo

[HERE](https://cry-player.autobiografia.repl.co)
## Run Locally

Clone the project

```bash
  git clone https://github.com/autobiografia/cry-player.git
```

Go to the project directory

```bash
  cd cry-player
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
